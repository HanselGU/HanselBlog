# 成都游玩体验

------

#### 第一站 到达

飞机上很无聊。

下降的时候，天府机场的天气条件不行，飞机盘旋了很久，最终转去了双流。好消息，双流离市区近一些。

酒店订在春熙路，四人电竞房，相当舒服了。



#### 第二站 三星堆

二号厅是展示修复的展厅我记得，不明觉厉。



#### 第三站 武侯祠

武侯祠，说实话不好看，感觉纯是为了情怀来看丞相的。唯一有段路叫做‘红墙’的，有点子感觉，但是很多人选择在这里驻足拍照，所以也没有什么好看。

<img src="/static/IMG_8661.JPG" alt="IMG_8661" style="width:500px" />

武侯祠后面是刘湘的墓，还挺大的，就是冷清。刘湘干了一辈子破事，最后能躺在这里，也算是不错了。

出来就是锦里，一堆店面，有一个树下是一个挂许愿签的地方，有新的，也有已经发霉的。给我的印象挺深刻。

最后买了两个纪念章送给表弟妹。



#### 总结

成都美食是真不错。唯一的就是老妈蹄花这个店到处都是，隔两百米就有一家，无敌了我只能说。

最后我们几个人总结了一下，开个玩笑说不如待在酒店里开黑。其实是很久没有几个人一起出去待着，怀念这种感觉罢了。
